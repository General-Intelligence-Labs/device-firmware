# Changelog — Egocentric headset

All notable changes to egocentric headset firmware are documented here. Format follows [Keep a Changelog](https://keepachangelog.com/); versioning is SemVer with the `ego-v` tag prefix and a source-SHA suffix (e.g. `ego-v0.1.0-524803e`).

## [ego-v0.1.0-524803e] — 2026-05-28

First public release.

- Initial firmware image for the egocentric headset.
- HTTP info endpoint reports `build: "524803e"`.
- Ships with the `gi-flash` single-binary flasher (Linux x86_64; tested on Ubuntu 24.04 LTS only — other distributions may work but are unvalidated).
